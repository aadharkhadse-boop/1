Encrypt monthly data

Turns a plain .xlsx report into an encrypted .enc file for the dashboard. Everything runs locally in this page — nothing is uploaded anywhere. Use the same password that unlocks the dashboard.

Name each source file YYYY-MM.xlsx (e.g. 2026-07.xlsx) — the encrypted file keeps the name as 2026-07.enc. Put the downloaded .enc into app/src/data/monthly/ in your repo, then commit & push.
